# DeposeGirls_AssetDecDL
用於下載與解密 放置少女 (Use to download all Depose Girls resources and decrypt files)

以前習慣用的net framework 4.7太舊了，剛好用一些新的.Net語法重新優化代碼 <br>
至於用法的話，跟之前差不多，就不多說了

另外一點要提的是Web選項的話，因為會先過DMM的連結抓個資訊，通常是不會擋，但少數時候可能會需要VPN <br>
以前Web好像是Cocos2d-JS的樣子，現在也改成Unity了，弄起來方便多了，至於Android選項的話，沒特別需求就不用下了
